## Issue 1: Flicker during sidebar resize
The component clears innerHTML before mermaid.render() completes, causing a flash of empty content.

Fix: Instead of clearing innerHTML before render, render into a detached div and swap in the SVG when ready. No flash:
```tsx
const hiddenDiv = document.createElement('div')
mermaid.render(renderId, chart).then(({ svg }) => {
  hiddenDiv.innerHTML = svg
  // Swap: replace container content only after SVG is ready
  if (containerRef.current && chart === renderedRef.current) {
    containerRef.current.innerHTML = hiddenDiv.innerHTML
  }
})
```

Also: during sidebar drag, skip re-render entirely — only render when width changes by more than 10px from last render:
```tsx
const lastRenderedWidth = useRef(0)
// In resize callback:
if (Math.abs(nextWidth - lastRenderedWidth.current) < 10) return
lastRenderedWidth.current = nextWidth
```
