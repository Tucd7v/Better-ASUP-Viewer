## Requirements
1. Column 1: Cluster name - clickable to expand/collapse nodes within
2. Column 2: Latest upload time for this cluster
3. Column 3: Node list - indented under cluster, each with file count badge
4. Column 4: Count of files uploaded in last 24 hours for this cluster
5. Global search bar: filters by cluster name OR node hostname (dropdown toggle)
6. Time filter: dropdown with options (Today, Last 7 days, Last 30 days, All)
