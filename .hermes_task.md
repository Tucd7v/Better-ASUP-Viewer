Two fixes for NodeHUD + FileTree node colors:

## Fix 1: NodeHUD columns — max 3 per column
Change `.hud-node-list` CSS from `flex-direction: column` to use CSS Grid or flex-wrap so max 3 nodes per column, then wraps to a new column.

CSS approach:
```css
.hud-node-list {
  display: grid;
  grid-auto-flow: column;
  grid-template-rows: repeat(3, auto);
  gap: 3px;
  ...
}
```

With grid-auto-flow: column, nodes fill down then overflow right into another column. 3 rows max.

Change FileTree to use `session.nodeColor` directly instead of index-based fallback:
