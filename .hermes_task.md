Add global font size control to Viewer toolbar for TXT/XML/EMS cards.

## 1. ViewerContext.tsx — add fontSize state
- Add `fontSize: number = 13` to initial state
- Add `SET_FONT_SIZE` action with payload `number`
- Export in context value

## 2. ViewerPage.tsx — toolbar control
Add in toolbar, between Grid button and spacer:
```
[-]  13  [+]
```
- [-]/[+] buttons change font size by 1px
- Range: 10-18, clamp both ends
- Read/Write via ViewerContext: `state.fontSize` / `dispatch({ type: 'SET_FONT_SIZE', fontSize: newSize })`
- Style: same as other toolbar buttons (monospace, 11px, border-radius 4)

## 3. TextFileCard.tsx — apply fontSize
- Read `state.fontSize` from useViewer()
- Apply to the `.txt-log-line` elements: `fontSize: state.fontSize || 13`

## 4. XMLFileCard.tsx — apply fontSize
- Apply to table cells (`td`): `fontSize: state.fontSize || 13`

## 5. EMSFileCard.tsx — apply fontSize
- Apply to log event lines: `fontSize: state.fontSize || 13`

## Build
Run `npm run build` from frontend/ to verify.
