Fix two XMLFileCard issues:

## Issue 1: Title + hostname not vertically centered
In the card header, the filename text and hostname text don't align vertically with the icon (📊) and buttons ([−] [×]).

Current header structure:
```
<div style={{ display: 'flex', alignItems: 'center', gap: 6, padding: '6px 10px', ... }}>
  <span>📊</span>
  <div style={headerTitleStyle}>
    <span style={filenameStyle}>{filename}</span>
    <span style={headerDividerStyle} />
    <span style={hostnameStyle}>{hostname}</span>
  </div>
  <button>[−]</button>
  <button>[×]</button>
</div>
```

The parent flex row has `alignItems: 'center'` which should align children. But `headerTitleStyle` has `display: flex, alignItems: center, flex: 1, minWidth: 0`. Its internal children (filename + hostname) should also center.

Check: does the filenameStyle or hostnameStyle have mismatched line-height or font-size causing visual misalignment? Try adding `lineHeight: '16px'` to both filenameStyle and hostnameStyle.

## Issue 2: ColSwapMenu position wrong on first render
symptom: The popup appears at wrong position when first opening an XML card, but position corrects after toggling grid mode on/off.

Root cause: `wrapperRef` measurement in `openSwapMenu` uses `getBoundingClientRect()` which depends on DOM layout. On first render, React-virtualized content may not be fully laid out. Grid mode toggle forces a re-layout.

Fix: wrap the position calculation in a `requestAnimationFrame` or `setTimeout(0)` to ensure DOM is settled before reading measurements.

```tsx
const openSwapMenu = (col: string, e: React.MouseEvent) => {
  e.stopPropagation()
  if (swapMenuCol === col) { setSwapMenuCol(null); return }
  const th = thRefs.current.get(col)
  const wrapper = wrapperRef.current
  if (th && wrapper) {
    requestAnimationFrame(() => {
      const thRect = th.getBoundingClientRect()
      const wRect = wrapper.getBoundingClientRect()
      setSwapMenuPos({
        left: thRect.left - wRect.left,
        top: thRect.bottom - wRect.top + 4,
      })
    })
  }
  setSwapMenuCol(col)
}
```
