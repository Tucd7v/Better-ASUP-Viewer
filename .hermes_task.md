Two features for NodeASUP frontend:

## 1. XML card: resizable column widths
Add draggable column border resize to XMLFileCard table headers. When user hovers over the right edge of a column header, cursor changes to `col-resize`. Dragging left/right adjusts that column's width.

Implementation:
- Track column widths in state: `{[col: string]: number}`
- Default width per column: `getColumnWidth(col)` which is already defined as `Math.max(100, col.length * 9)`
- On header `<th>`, add a `<div>` on the right edge (width 4px, cursor col-resize)
- On mousedown: record startX and current column width
- On mousemove: calculate new width = Math.max(50, startWidth + (e.clientX - startX))
- On mouseup: stop tracking
- Use document-level event listeners so dragging works outside the th bounds
- Apply width via `style={{ width: widths[col] || getColumnWidth(col) }}` on th and td

## 2. Grid mode: max 8 cards, 2 columns
In `ViewerPage.tsx`, the SplitGrid component has a max cards limit. Change from 4 to 8. Use CSS Grid with 2 columns, cards stack vertically when > 2.

Implementation:
- Find the `MAX_CARDS` or similar constant for SplitGrid
- Change from 4 to 8
- CSS: `grid-template-columns: repeat(2, 1fr)` with `grid-auto-rows: 1fr`
- Remove `grid-template-rows: repeat(2, 1fr)` — let it auto-flow
- Adjust the divider drag logic for 2-col layout (vertical divider between columns)
