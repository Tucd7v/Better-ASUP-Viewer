## Changes

### 1. Remove ColSwapMenu
- Delete the ColSwapMenu component entirely

### 2. Add column pinning (sticky lock)
- Add `pinnedCols: Set<string>` state
- Replace the old ⇄ button with a 🔒 button on each column header
- When clicked, toggle the column in pinnedCols
- Pinned columns render with `position: sticky; left: 0; z-index: 1; background: inherit` so they stay fixed during horizontal scroll
- The first N columns (pinned ones) should visually stack from the left edge

### 3. Keep drag-to-reorder
- Don't touch the draggable/dragStart/dragEnter/dragEnd handlers
- Column reordering via drag should still work
