# Add resizable dividers between SplitMode cards (like iPad Split View)

## Current state
SplitMode uses CSS Grid with `1fr` units — all cards get equal space, no way to resize.

## Goal
Add draggable dividers between cards so users can resize card areas. Behavior depends on card count:

- **1 card**: no divider (full canvas)
- **2 cards**: 1 vertical divider between left/right
- **3 cards**: 2 vertical dividers (3 columns)
- **4 cards**: 1 horizontal + 1 vertical divider (cross in center of 2x2)

## Implementation approach

Replace the CSS Grid with a flex-based layout that uses percentage-based sizing.

### State
```ts
const [splitRatios, setSplitRatios] = useState<number[]>([50, 50]) // percentages
```

### Layout
Instead of CSS Grid, use nested flex containers:

```
2 cards: 
  <div style={{display:'flex', width:'100%', height:'100%'}}>
    <card style={{width: splitRatios[0]+'%'}} />
    <divider onMouseDown={startDrag} />  // 4px wide, cursor: col-resize
    <card style={{width: splitRatios[1]+'%'}} />
  </div>

3 cards:
  Similar but 3 cards + 2 dividers

4 cards:
  <div style={{display:'flex', flexDirection:'column'}}>
    <div style={{display:'flex', height: vRatio+'%'}}>  // top row
      <card /><divider /><card />
    </div>
    <divider horizontal />  // 4px tall, cursor: row-resize
    <div style={{display:'flex', height: (100-vRatio)+'%'}}>  // bottom row
      <card /><divider /><card />
    </div>
  </div>
```

### Divider component
```tsx
const Divider = ({ direction, onMouseDown }: {direction: 'h'|'v', onMouseDown: ...}) => (
  <div style={{
    width: direction==='v' ? 4 : '100%',
    height: direction==='h' ? 4 : '100%',
    cursor: direction==='v' ? 'col-resize' : 'row-resize',
    background: '#e2e8f0',
    flexShrink: 0,
    transition: 'background 0.15s',
  }}
    onMouseEnter={e => e.currentTarget.style.background='#3b82f6'}
    onMouseLeave={e => e.currentTarget.style.background='#e2e8f0'}
    onMouseDown={onMouseDown}
  />
)
```

### Drag logic for dividers
On mousedown, track mouse position. On mousemove, calculate new ratio. On mouseup, stop tracking. Use `useRef` and `useState` for the drag state.

For 2-card layout: drag divider horizontally → update horizontalRatio (0-100)
For 3-card layout: drag either divider → update both ratios proportionally  
For 4-card layout: drag vertical divider → update vSplit; drag horizontal divider → update hSplit

### Minimum sizes
Each card should have a minimum of 20% width/height to prevent collapsing.
