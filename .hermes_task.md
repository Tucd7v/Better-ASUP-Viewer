Fix XML column header button layout in XMLFileCard.tsx.

## Problem
- Buttons (sort ▲/▼, swap ⇄, lock 🔒) overlap with column title text
- Button spacing is inconsistent (sort at right:48 is too far from swap at right:24)

## Required layout
All three buttons grouped together on the right side, with equal spacing (8px gap between each):

```
[col name text.........]  [sort ▲] [swap ⇄] [lock 🔒] [resize|]
                            ^-8px-^ ^-8px-^ ^-8px-^
```

### Implementation
1. Change `<th>` padding-right from 46px to a value that accommodates all 3 buttons + gaps: about 72px (3 buttons × ~16px + 3 gaps × 8px)
2. Position buttons from right edge:
   - Lock button (🔒): `right: 4`
   - Swap button (⇄): `right: 24` (4 + 16 + 4)
   - Sort button (▼): `right: 44` (24 + 16 + 4)
3. All three buttons share same vertical centering: `position: absolute; top: 50%; transform: translateY(-50%)`
4. Each button width should be ~16px with text centering (`fontSize: 10, padding: '1px 2px'`)
5. Column title text must NOT overlap with buttons — text has `overflow: hidden; textOverflow: ellipsis; whiteSpace: nowrap` which should handle this naturally since buttons are absolutely positioned on the right
6. Keep resize handle at `right: -2`
7. Don't break column resize, drag-to-reorder, or pinned column sticky positioning
