# Add Tab System + AI Autonomous Mode Tab to NodeASUP Viewer

## Goal
Add a browser-style tab bar to the viewer, allowing multiple independent canvases. When AI autonomous mode is triggered, it opens in its own tab.

## Architecture

### Current state
```
ViewerInner
  state: nodes[], edges[]        // single canvas
  → ReactFlow renders nodes
  → handles interactions
```

### New state
```
ViewerInner
  tabs: Tab[]                    // array of tab objects
  activeTabId: string            // which tab is active
  → each tab has: { id, name, nodes, edges }
  → ReactFlow uses key={activeTabId} to remount on tab switch
```

### Tab data structure
```ts
interface Tab {
  id: string        // uuid
  name: string      // display name
  nodes: Node[]     // independent canvas nodes
  edges: Edge[]     // independent canvas edges
  isAutoAI?: boolean // true for AI autonomous mode tabs
}
```

## Implementation

### 1. TabBar component (NEW file: `frontend/src/features/viewer/TabBar.tsx`)

A horizontal bar above the template bar showing tabs. Each tab shows its name and a [×] close button. Right side has a [+] button to add new empty tabs.

```tsx
// Simplified:
<div className="tab-bar nodrag" style={{ display: 'flex', background: '#f1f5f9', borderBottom: '1px solid #e2e8f0' }}>
  {tabs.map(tab => (
    <div key={tab.id} style={{ 
      padding: '4px 12px', cursor: 'pointer',
      background: tab.id === activeTabId ? '#fff' : 'transparent',
      borderBottom: tab.id === activeTabId ? '2px solid #3b82f6' : '2px solid transparent',
    }} onClick={() => setActiveTabId(tab.id)}>
      <span>{tab.name}</span>
      {tabs.length > 1 && <span onClick={e => { e.stopPropagation(); closeTab(tab.id) }}> ×</span>}
    </div>
  ))}
  <button onClick={addTab} style={{ marginLeft: 'auto', padding: '4px 8px' }}>+</button>
</div>
```

### 2. Refactor ViewerPage.tsx (major changes)

**Replace single nodes/edges state with tabs:**

Remove:
```ts
const [nodes, setNodes, onNodesChange] = useNodesState([])
const [edges, setEdges, onEdgesChange] = useEdgesState([])
```

Add:
```ts
const [tabs, setTabs] = useState<Tab[]>([{ id: 'tab-1', name: 'View 1', nodes: [], edges: [] }])
const [activeTabId, setActiveTabId] = useState('tab-1')
```

**Helper functions:**

```ts
const activeTab = tabs.find(t => t.id === activeTabId)!

const updateActiveTab = (updater: (tab: Tab) => Tab) => {
  setTabs(prev => prev.map(t => t.id === activeTabId ? updater(t) : t))
}

const addTab = () => {
  const id = `tab-${Date.now()}`
  setTabs(prev => [...prev, { id, name: `View ${prev.length + 1}`, nodes: [], edges: [] }])
  setActiveTabId(id)
}

const closeTab = (id: string) => {
  setTabs(prev => {
    const next = prev.filter(t => t.id !== id)
    if (id === activeTabId && next.length > 0) {
      setActiveTabId(next[next.length - 1].id)
    }
    return next
  })
}
```

**Replace all setNodes/setEdges calls:**

Every place that calls `setNodes(...)` or `setEdges(...)` must be updated to use `updateActiveTab`. For example:

Before:
```ts
setNodes(prev => [...prev, newNode])
setEdges(prev => [...prev, newEdge])
```

After:
```ts
updateActiveTab(tab => ({
  ...tab,
  nodes: [...tab.nodes, newNode],
  edges: [...tab.edges, newEdge]
}))
```

**Key places to update (search for `setNodes` and `setEdges`):**
- `handleFocusFile` — adds cards to canvas
- `handleSaveTemplate` — reads nodes/edges from active tab
- `handleLoadTemplate` — writes nodes/edges to active tab
- Clear button (`setNodes([]); setEdges([])`)
- `onConnect` callback
- `onEdgeDoubleClick` label editing

**ReactFlow key:**
```tsx
<ReactFlow key={activeTabId} ...>
```
This ensures React remounts ReactFlow when switching tabs, giving each tab a clean canvas state.

### 3. AI Autonomous Mode Tab

In `AIChatPanel.tsx`, when the user switches to autonomous mode:

```tsx
// When mode switches to 'autonomous':
useEffect(() => {
  if (mode === 'autonomous') {
    // Create a new AI tab if one doesn't exist
    const aiTab = tabs.find(t => t.isAutoAI)
    if (!aiTab) {
      const newTab = { id: `ai-${Date.now()}`, name: 'AI Analysis', nodes: [], edges: [], isAutoAI: true }
      setTabs(prev => [...prev, newTab])
      setActiveTabId(newTab.id)
    } else {
      setActiveTabId(aiTab.id)
    }
  }
}, [mode])
```

When AI sends its analysis results with file references, those files should be opened as cards in the AI tab.

### 4. Persist state across renders

The `fileMetaRef` and session data remain global (not per-tab) since they're data from the backend. Only the canvas state (nodes/edges) is per-tab.

### 5. Template system

Templates save/load from the active tab's nodes/edges. No structural change needed — just use `activeTab.nodes` instead of `nodes`.
