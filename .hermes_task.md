Add drag-to-reorder for cards in Grid mode (SplitGrid, ViewerPage.tsx).

## Behavior
- User drags a card by its header
- Drags over another card position
- On drop, the two cards swap positions
- Visual feedback: dragged position shows highlight border

## Implementation
### SplitCard component
- Make card `<div>` draggable with `draggable` attribute
- Add `onDragStart`: store source index in dataTransfer
- Add `onDragOver`: `e.preventDefault()` to allow drop
- Add `onDrop`: read source index, call swap callback

### SplitGrid
- Track drag source index in state
- Highlight target zone with border (already has dragOverZone for file drops)
- On drop: call `onSwapCard(sourceIdx, targetIdx)` 

### ViewerInner
- Add `onSwapCard` handler to SplitGrid props
- Swap two nodes in the active tab's nodes array
- Use `setNodesForTab` to update

## Edge cases
- Don't allow dropping onto the same position
- Don't interfere with existing file-drop-to-replace behavior (dragOverZone)
- Grid mode only (canvas mode has its own drag behavior)

## Build
Run `npm run build` from frontend/ to verify.
