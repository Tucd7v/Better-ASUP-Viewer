Fix AI chat panel ReactMarkdown code blocks to correctly render ASCII topology diagrams and network layouts.

## Problem
When AI outputs ASCII network topology diagrams in code blocks, they lose formatting — whitespace collapses and the diagram breaks.

## Fix in AIChatPanel.tsx
Add custom `pre` and `code` components to ReactMarkdown's `components` prop (around line 276):

### pre component
```
pre: ({ children }) => (
  <pre style={{
    background: '#1e293b',
    color: '#e2e8f0',
    padding: 12,
    borderRadius: 6,
    overflowX: 'auto',
    fontSize: 11,
    fontFamily: 'ui-monospace, Consolas, monospace',
    lineHeight: 1.3,
    whiteSpace: 'pre',       // CRITICAL — preserves ASCII diagram alignment
  }}>{children}</pre>
)
```

### code component
```
code: ({ className, children, ...props }) => {
  const isLang = /language-(\w+)/.exec(className || '')
  if (isLang) {
    return <code className={className} {...props}>{children}</code>
  }
  // Inline code: light background
  return <code style={{ background: '#f1f5f9', padding: '2px 5px', borderRadius: 3, fontSize: 12, fontFamily: 'monospace' }} {...props}>{children}</code>
}
```

Add both alongside the existing `a` link component.

## Key
- `whiteSpace: 'pre'` on `<pre>` is the critical fix — without it, ASCII art collapses
- `overflowX: 'auto'` for wide topology diagrams
- Keep existing `remarkGfm` plugin
- Keep existing `target="_blank"` on links

## After fix
Ask AI to draw a network topology — the ASCII art in code blocks should display correctly with preserved spacing.
